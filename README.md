# youtube-lyrics-video

## config.json
```json
{
    "credentials" : {
        "email": "", 
        "pass": ""
    },
    "SpotifyCookie": "",
    "TrackList": [
        { "trackId": "", "title": "", "artist": "" },
    ]
}
```
```credentials in your Google account```  
```TrackList.trackId in spotify track id```     
```SpotifyCookie in spotify sp_dc cookie```
